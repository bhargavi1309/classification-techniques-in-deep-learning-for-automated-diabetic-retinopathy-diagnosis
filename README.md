# Classification Techniques in Deep Learning for Automated Diabetic Retinopathy Diagnosis

## Overview
This project uses deep learning techniques to automatically detect diabetic retinopathy from retinal fundus images.

## Features
- Image preprocessing
- CNN-based classification
- Multi-stage disease detection
- Performance evaluation

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Pandas
