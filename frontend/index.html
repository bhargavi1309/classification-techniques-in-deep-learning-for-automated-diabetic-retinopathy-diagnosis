About RetinaAI

Deep learning for automated Diabetic Retinopathy diagnosis

🧠

Deep Learning Classifier

Uses an EfficientNet-B4 architecture pre-trained on ImageNet and fine-tuned on the EyePACS and APTOS-2019 datasets (88,000+ retinal images) for 5-class DR grading.

🔬

Grad-CAM Visualisation

Gradient-weighted Class Activation Mapping highlights the retinal regions (microaneurysms, exudates, neovascularisation) that most influenced the model's decision.

📊

Severity Grading Scale

Follows the International Clinical Diabetic Retinopathy (ICDR) 5-point scale: Grade 0 (No DR) through Grade 4 (Proliferative DR), matching clinical practice guidelines.

✅

Clinical Performance

The model achieves AUC-ROC 0.97 for referable vs. non-referable DR detection on the MESSIDOR-2 benchmark, comparable to expert ophthalmologists.

DR Severity Reference

Grade Label Key Features Recommended Action
0 No DR No abnormalities Annual routine screening
1 Mild NPDR Microaneurysms only 12-month follow-up
2 Moderate NPDR Dot/blot haemorrhages, exudates Refer within 6 months
3 Severe NPDR 4-2-1 rule, IRMA Refer within 1 month
4 Proliferative DR Neovascularisation, vitreous haemorrhage URGENT referral
